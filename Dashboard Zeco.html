Manual de Instrucciones

👨‍💻  Este programa ha sido realizado por David Gutiérrez Pastor.
Si te gusta, puedes decírselo... ¡prometemos que no cobra por sonreír! 😄
  • Carga de datos
    En la pantalla de inicio, arrastra el fichero CSV sobre la zona de carga o haz clic en ella para seleccionarlo. Si ya existe una sesión guardada aparecerá el botón Ver sesión actual para recuperarla sin necesidad de cargar de nuevo.
    El icono de cuadrícula (visible solo en la landing) activa el modo de asignación manual de columnas. Úsalo si el CSV tiene un formato no reconocido. Una vez activo, se ilumina en dorado.
    Para cargar un nuevo CSV desde el dashboard usa el icono de la barra de filtros.
  • Proceso automático (macro SAP)
    La macro VBA extrae datos de SAP (transacciones ZECO y ZECO_REP), genera el CSV con pendientes y datos anuales, y abre el dashboard automáticamente. No es necesaria ninguna exportación manual.
  • Panel de métricas y filtros activos
    El panel ocupa una fila completa dividida en dos bloques: Filtros Activos (izquierda, borde azul) y Métricas Globales (derecha, borde verde). Las etiquetas de filtros aplicados aparecen dentro del bloque izquierdo — pulsa × en cada etiqueta para eliminarlo.
    📅 ECTs (últimos 12 meses): total del último año.   📋 ECTs Pendientes: total actual.   📈 Pico Temporal: mes con más ECTs.
    🏆 Proyecto Crítico: proyecto con más ECTs pendientes. Haz clic en la tarjeta para ver el ranking completo ordenado. Desde el ranking, pulsa cualquier proyecto para ver su lista de ECTs con número y fecha. Usa ↗ para abrir en Windchill o el icono para copiar todos los números al portapapeles.
  • Filtros
    Filtra por proyecto con el selector o por rango de fechas. Las métricas y el gráfico izquierdo se actualizan al instante. Haz clic en una barra del gráfico para filtrar ese mes directamente.
    Icono de papelera — limpia todos los filtros activos de una vez.
  • Gráficos
    📊 Izquierda — ECTs pendientes: refleja los filtros activos. El botón + Anual superpone la tendencia anual (verde discontinuo) sobre los pendientes (azul), en 2D y 3D. El tooltip muestra ambos valores. Pulsa de nuevo para desactivar.
    📅 Derecha — Tendencia Anual: últimos 12 meses del CSV anual, independiente de los filtros.
    Cada gráfico tiene controles 2D/3D y tipo Barras / Líneas / Circular. En 3D: arrastrar para rotar, botón derecho para desplazar, rueda para zoom.
  • Tabla de ECTs
    Lista el conjunto filtrado con número, fecha, proyecto y descripción. Navega con la paginación inferior.
    ↗ Abre en Windchill y copia el ID.   Copia todos los IDs visibles.   Exporta las filas a Excel.
